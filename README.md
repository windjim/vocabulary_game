# CNN Vocabulary Game

一個互動式的英文單字測驗應用程式，支援 Excel 檔案上傳自訂單字庫。

## 功能特色

- **Excel 上傳功能**：支援上傳 .xlsx/.xls 格式的單字檔案
- **多種出題模式**：
  - 單選模式：選擇單一單元出題
  - 多選模式：選擇多個單元依序出題
  - 混合出題：從多個單元中隨機出題
  - 只出不確定：針對標記為「不確定」的單字重複練習
- **即時統計**：顯示得分、題數、正確率
- **例句顯示**：每個單字可包含例句輔助學習
- **本地儲存**：使用 localStorage 儲存單字資料和學習進度

## 使用方式

1. 開啟 [index.html](index.html) 或訪問線上版本
2. 點擊「上傳 Excel」按鈕，上傳你的單字檔案
3. 選擇出題模式和單元
4. 點擊「開始出題」開始測驗

## Excel 檔案格式

Excel 檔案應包含以下欄位（使用繁體中文標題）：

| 單字 | 詞性 | 意思 | 例句 |
|------|------|------|------|
| example | n. | 例子 | This is an example. |

- **必填欄位**：單字、意思
- **選填欄位**：詞性、例句
- **頁籤名稱**：每個頁籤的名稱會作為單元名稱（例如：L1, L2, L3...）

## 技術規格

- 純前端應用程式（HTML + JavaScript）
- 使用 [SheetJS](https://sheetjs.com/) 解析 Excel 檔案
- 響應式設計，支援行動裝置
